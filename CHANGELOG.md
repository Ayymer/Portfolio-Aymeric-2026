# Changelog

All notable changes to this project will be documented in this file.

## [1.0.1] — 2026-03-23

### Added
- TiCO case study with full sections and image assets
- Image-only section variant support in content schema
- Email mailto button in case study pages
- Custom AF monogram favicon (replaces default Astro favicon)

## [1.0.0] — 2026-03-22

### Added
- Portfolio live on Infomaniak (eco-friendly Swiss hosting)
- GitHub Actions auto-deploy pipeline (push to `main` → FTP deploy)
- Lounge Cruise case study with full sections and image compositions
- Halftone dot shader effect on About page (paper-design/shaders)
- Umami analytics integration (privacy-friendly, cookie-free)
- Code audit fixes: image optimization, font pruning, design token usage, component extraction

### Stack
- Astro 6, Tailwind CSS 4, Bun
- TWK Lausanne + IBM Plex Mono fonts
- Content collections with Zod schema validation

### Infrastructure
- Hosted on Infomaniak (100% renewable energy, Swiss datacenter)
- CI/CD via GitHub Actions → FTP
- Analytics via Umami Cloud
